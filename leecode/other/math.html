求模运算

如果有a%m = c%m和b%m = d%m，那么有以下模运算成立

  • (a + b) % m = (c + d) % m
  • (a - b) % m = (c - d) % m
  • (a x b) % m = (c x d) % m

注意上述三个没有除法运算。。另外由于最终结果可能会溢出，故需要使用更大范围的类型来保存求模之前的结果。另外若a是负数时往往需要改写为a%m+m，这样就保证结果在[0, ,m-1]范围内了。

快速幂运算

与剑指offer中数值的整数次方的思路基本一致

public class Solution {
    public double Power(double base, int exponent) {
        // 利用移位运算的另一种写法
        // 为了防止幂运算移出，在这里取模max_value
        mod = Double.MAX_VALUE;
        double res=1.0 % mod;
        long p = Math.abs(exponent);
        if (p < 0 && base == 0)
            throw new RuntimeException("分母不能为0");
        else if (p == 0)
            return 1;

        while (p != 0) {
            if ((p & 1) == 1)
                res = res * base % mod;
            base = base * base % mod;
            p >>= 1;
        }
        return exponent > 0 ? res : 1/res;
  }
}

最大公约数(GCD, Greatest Common Divisor)

常用的方法为辗转相除法，也称为欧几里得算法。不妨设函数gcd(a, b)是自然数a, b的最大公约数；不妨设a > b，则有a = b*p+q，那么对于gcd(b, q)则是b和q的最大公约数，也就是说gcd(b, q)即能整除b，又能整除a,如此反复最后得到gcd(a, b) = gcd(c, 0)，第二个整数为0时直接返回c

public static long gcd(long a, long b) {
  return (b==0) ? a : gcd(b, a%b)
}

最小公倍数LCM和最大公约数之间的关系为 LCM(a, b)* gcd(a, b)=|ab|

扩展欧几里得算法

求解整数系数x和y满足d = gcd(a, b) = ax + by, 仿照欧几里得算法，应该要寻找$gcd(b, a\%b) = bx_1 + (a\%b)y_1$

public class Solution {
  public static int[] gcdExt(int a, int b) {
    if (b == 0) {
      return new int[] {a, 1, 0};
    }
    else {
      int[] vals = gcdExt(b, a%b);
      int d = vals[0];
      int x = vals[2];
      int y = vals[1];
      y -= x*(a / b);
      return new int[] {d, x, y};
    }
  }
  public static void main(String[] args) {
    int a = 4, b = 8;
    int[] result = gcdExt(a, b);
    System.out.printf("d = %d, x = %d, y = %d.\n", result[0], result[1], result[2]);
  }
}

problem

判断是否存在整数x和y使得ax + by = 1，若有，则输出

Solution

不妨设gcd(a, b)= M，那么有$M(a_1x + b_1y) = 1$ ==> $a_1x + b_1y = 1/M$；如果M大于1，由于等式左边为整数，故等式不成立，所以要想上述等式有解，必有gcd(a, b) = 1

对上面的代码的返回结果加一个if else判断，即可得到当前问题的解.

素性测试

用来判断是否是素数

如果d是n的约数，则易知$n = d * \frac{n}{d}$，因此$\frac{n}{d}$也是n的约数，且这两个约数中的较小者min(d, n/d) <= $\sqrt{n}$.

说白点就是 a * b = c，则a、b都是c的约数，则min(a, b) <= $\sqrt{n}$

public static boolean isPrime(int n) {
  for (int i=2; i*i <= n; i++) {
    if (n % i) == 0
      return false;
  }
  return n != 1;
}

埃氏筛法

用来查找[1, n)的所有素数

素性测试针对的是单个整数，如果需要枚举整数n以内的素数就需要埃氏筛法了。核心思想是枚举从小到大的素数并将素数的整数倍依次从原整数数组中删除，余下的即为全部素数。

public static List<Integer> sieve(int n) {
  List<Integer> prime = new ArrayList<Integer>();
  boolean[] isPrime = new boolean[n+1];
  Arrays.fill(isPrime, true);
  isPrime[0] = false;
  isprime[1] = false;
  for (int i=2, i<=n; i++) {
    if (isPrime[i]) {
      prime.add(i);
      for (int j=2*i, j<=n; j+=i) {
        isPrime[j] = false;
      }
    }
  }
  return prime;
}

区间筛法

求区间[a, b)内有多少素数？

埃氏筛法得到的是[1, n)内的素数，求区间素数时不太容易直接求解，我们采取以退为进的思路先用埃氏筛法求得[1, b)内的素数，然后截取为[a, b)即可。

  Arrays.fill(isPrime, true);
  isPrime[0] = false;
  isPrime[1] = false;
  for (int i=2, i<b; i++) {
    if (isPrime[i]) {
      if (i >= a)
        prime.add(i);
      for (int j=2*i; j<b; j+=i) {
        isPrime[j] = false;
      }
    }
  }
  return prime;
}

public static Map<Integer, Integer> getPrimeFactor(int n) {
        Map<Integer, Integer> result = new HashMap<Integer, Integer>();
        for (int i = 2; i * i <= n; i++) {
            // if i is a factor of n, repeatedly divide it out
            while (n % i == 0) {
                if (result.containsKey(i)) {
                    result.put(i, result.get(i) + 1);
                } else {
                    result.put(i, 1);
                }
                n = n / i;
            }
        }
        // if n is not 1 at last
        if (n != 1) result.put(n, 1);
        return result;
