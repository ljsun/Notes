bitmap

参考链接1

简介

所谓的BitMap就是用一个bit位来标记某个元素所对应的Value；由于采用了bit为单位来存储数据，因此在存储空间方面，可以大大节省。

基本思想

用一个具体的例子来讲解，假设要对0~7内的5个元素（4, 7, 2, 5, 3）排序（假设没有重复元素）。那么我们可以采用BitMap的方法来达到排序的目的。要表示8个数，就只需要8个bit（1Bytes）

  1. 首先我们开辟1字节（8bit）的空间，将这些空间的所有bit位都置为0

  2. 然后遍历这5个元素，首先第1个元素是4，那么就把4对应的位置位1，如下图

​ 然后处理第2个元素7，一直到处理完所有的元素，将相应的位置为1，这时候内存的bit位的状态如下：

     3. 然后我们再遍历一遍bit区域，将该位是1的位的编号输出（2, 3, 4, 5, 7），这样就达到了排序的目的

算法的思想比较简单，但关键是如何确定十进制的数如何映射到二进制bit位的map图（即十进制的数如何用二进制的bit位表示）

Map映射

假设需要排序或者查找的总数 N = 10000000；BitMap中1bit代表一个数字，1个int=32bit，那么N个数需要N/32 int空间。所以我们需要申请内存空间的大小为int a[1 + N/32]，其中：a[0]在内存中占32位，可以代表十进制数0~31，依次类推: BitMap表为:

    a[0]  --------->  0-31 
    a[1]  --------->  32-63 
    a[2]  --------->  64-95 
    a[3]  --------->  96-127 
    ..........

那么十进制数如何转换为对应的bit位，下面介绍用位移将十进制转换为对应的bit位

例如：

十进制1在a[0]中，位置如下图：

十进制31 在a[0]中，位置如下图:

十进制32 在a[1]中，位置如下图:

十进制33 在a[1]中，位置如下图:

通过上图分析得出通过以下几步将十进制如何转换为对应的bit位：

（1）求十进制数在对应数组a中的下标

十进制数0~31，对应在数组a[0]中；32~63对应在数组a[1]中；64~95对应在数组a[2]中 ......

分析得出：对于一个十进制数n，对应在数组a[n/32]中

例如n=11，那么n/32=0，则11对应在数组a中的下标为0；n=32，那么n/32=1，则32对应在数组a中的下标为1；n=106，那么n/32 = 3，则106对应数组a中的下标为3.

（2）求十进制数在对应数组a[i]中的下标

例如十进制数1在a[0]的下标为1，十进制数31在a[0]中的下标为31，十进制数32在a[1]中的下标为0。

分析得出：对于一个十进制数n，对应在数组a[n/32][n%32]中

（3）移位

对于一个十进制数n，对应在数组a[n/32][n%32]中，但数组a毕竟不是一个二维数组，我们通过移位操作来实现置1。

a[n/32] |= 1 << (n%32)【假设n%32=3，那么就是1左移3位，然后与a[n/32]进行“与”操作】

利用移位操作表示就是：

a[n>>5] |= 1 << (n & 0x1F)【其中n & 0x1F保留n的后五位，相当于求 n%32 】

import java.util.*;

public class BitMap {
    int[] a = new int[5];

    private void bitMap(int n) {
        // row = n / 32求十进制数在数组a中的下标
        int row = n >> 5;
        // n & 0x1F 保留n的后5位，其实就是相当于求 n % 32
        a[row] |= 1 << (n & 0x1F);
    }

    private int isExists(int n){ //判断所在的bit位是否为1
        // 返回值为0或者是1
        int row = n >> 5;
        return a[row] & (1 << (n & 0x1F));
    }

    private void show(int row) {
        System.out.println("BitMap位图展示：");
        for (int i=0; i<row; i++) {
            ArrayList<Integer> array = new ArrayList<Integer>();
            int temp = a[i];
            for (int j=0; j<32; j++) {
                array.add(temp & 1);
                temp >>= 1;
            }
            System.out.print("a[" + i+"" + "]->");
            for (int k=array.size()-1; k>=0; k--)
                System.out.print(array.get(k)+" ");
            System.out.println();
        }
    }

    public static void main(String[] args) {
        BitMap bitMap = new BitMap();
        int[] num = {1,5,30,32,64,56,159,120,21,17,35,45};
        for (int i=0; i<12; i++) {
            bitMap.bitMap(num[i]);
        }
        bitMap.show(5);
    }
}

应用范围：快速查找、去重、排序、压缩数据等

具体应用

已知某个文件内包含一些电话号码，每个电话号码为8位数字，统计不同号码的个数.

思路： 电话号码每个为8位数字，最小电话号码为00 000 000，最大为99 999 999，一共100 000 000个电话号码，100 000 000bit = 12 500 000Bytes = 12MBytes，故需要12MBytes的内存即可存储所有的电话号码

下面这段代码重点记住的是：

  1. 给定一组数，建立针对这组数的Bitmap【首先获取最大值max与最小值min，确定这组数可能的个数max-min+1；然后构建BitMap数组大小为(max-min+1)/32；然后将每一个数字在Bitmap中的对应位置为1（BitMap[(num-min)/32] |= （1 << (num-min)%32））】
import java.util.Random;

public class Phone {

    final int NUM = 1000;
    final int MIN = 10000000;
    final int MAX = 99999999;
    final int N = MAX - MIN + 1;

    private void setBitMap(int[] bitMap, int num) {

        num -= MIN;

        bitMap[num>>5] |= (1 << (num%32));
    }

    private int phoneCount(int[] phone, int n, int[] bitMap) {
        for (int i=0; i<n; i++) {
            setBitMap(bitMap, phone[i]);
        }
        // 统计个数【直接统计bitmap中有多少个1】
        int count = 0;
        for (int i=0; i<bitMap.length; i++){
            int temp = bitMap[i];
            while (temp > 0) {
                count += (temp & 1);
                temp = temp >> 1;
            }
        }
        return count;
    }
    public static void main(String[] args) {
        Phone phone = new Phone();
        int[] p = new int[phone.NUM];
        // 初始化用来存储电话号码的位图
        int[] bitmap = new int[phone.N/32 + 1];
        // 随机生成NUM个电话号码
        Random random = new Random();
        for (int i=0; i<phone.NUM; i++) {
            p[i] = random.nextInt(99999999) % phone.N + phone.MIN;
            System.out.println(p[i]);
        }
        System.out.println("电话号码个数->" + phone.phoneCount(p, phone.NUM, bitmap));
    }
}
